🚦 Smart Traffic Navigator

📌 Project Description

Smart Traffic Navigator is an intelligent system that predicts traffic conditions and suggests the shortest and fastest routes using Machine Learning and Deep Learning techniques.

This project helps users avoid traffic congestion and reach their destination efficiently.
🎯 Features

🚗 Shortest path navigation
📊 Traffic prediction using ML models
🧠 Deep learning-based decision making
📍 Start and destination input
⏱️ Displays distance, time, and traffic intensity
🌐 Interactive UI using Streamlit

🛠️ Tech Stack

Python 🐍
Streamlit (Frontend)
NumPy & Pandas (Data processing)
Scikit-learn (Machine Learning)
TensorFlow/Keras (Deep Learning)
Matplotlib (Visualization)
⚙️ How It Works

User enters start and destination points

System calculates possible routes

ML model predicts traffic conditions

Best route is selected based on:

Shortest distance
Least traffic
Minimum travel time
